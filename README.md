# 8-Bit Computer Simulator

A Python-based 8-bit computer simulator that emulates basic computer operations and instructions.

## Files

- `simulator.py` - Main simulator implementation
- `app.py` - Application interface
- `programs.py` - Sample programs and instruction sets
- `instructions.txt` - Documentation of available instructions

## Usage

Run the simulator:
```bash
python app.py
```

Or use the simulator module directly:
```bash
python simulator.py
```

## Features

- 8-bit instruction set architecture
- Basic arithmetic and logical operations
- Memory management
- Program execution simulation

## Requirements

- Python 3.x

## Getting Started

1. Clone the repository
2. Run `python app.py` to start the simulator
3. Load programs or write your own using the instruction set

For detailed instructions, see `instructions.txt`.
